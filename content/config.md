# 站点配置

修改下方 JSON 代码块即可更新站点信息，修改后重新构建生效。

| 字段 | 说明 |
|------|------|
| `authorName` | 作者名称，显示在头像下方 |
| `githubId` | GitHub 用户纯数字 ID，[点此获取](https://api.github.com/users/你的用户名) |
| `githubUsername` | GitHub 用户名，头像点击跳转 |
| `avatarSize` | 头像尺寸（像素） |
| `motto` | 个人签名，显示在头像区域 |

```json
{
  "authorName": "秋灯",
  "githubId": "85886906",
  "githubUsername": "qiudeng7",
  "avatarSize": "256",
  "motto": "喜欢写文章."
}
```
